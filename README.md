# HSP-Website

A dark terminal-style developer portfolio built with Vite, React, and regular CSS.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Current features

- Dark, terminal-inspired homepage layout
- Large lowercase name header
- Left-side info section with about, contact, tech stack, and site details
- Right-side profile image placeholder
- Download resume button
- Projects section with clickable project cards
- Simple GitHub activity section
- Pixel cat sprite animation planned in `public/Cat Sprite Sheet.png`
